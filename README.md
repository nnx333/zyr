# Currency Converter

**Автор:** Зырянов Вадим

## Описание
GUI-приложение для конвертации валют с использованием актуальных курсов через API exchangerate-api.com, сохранением истории операций и дружественным интерфейсом.

## Функциональность
- Конвертация между различными валютами
- Автоматическое получение актуальных курсов валют
- Сохранение истории конвертаций в JSON
- Возможность очистки истории
- Валидация ввода (проверка на положительные числа)
- Интуитивно понятный графический интерфейс
- Быстрая смена направлений конвертации

## Как получить API-ключ

1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com)
2. Нажмите "Get Free Key"
3. Зарегистрируйтесь, указав email
4. Подтвердите email
5. Получите API-ключ на странице дашборда
6. Замените `YOUR_API_KEY_HERE` в файле `currency_converter.py` на ваш ключ:

self.api_key = "your-api-key-here"

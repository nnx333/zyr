# Currency Converter

**Автор:** Зырянов Вадим

## Описание
GUI-приложение для конвертации валют с использованием актуальных курсов через API exchangerate-api.com, сохранением истории операций и дружественным интерфейсом.

## Функциональность
- Конвертация между различными валютами
- Автоматическое получение актуальных курсов валют
- Сохранение истории конвертаций в JSON
- Возможность очистки истории
- Валидация ввода (проверка на положительные числа)
- Интуитивно понятный графический интерфейс
- Быстрая смена направлений конвертации

## Примеры использования

### 📝 Пример 1: Базовая конвертация валют

**Задача:** Узнать, сколько евро вы получите за 100 долларов США.

**Пошаговые действия:**

1. Запустите приложение Currency Converter
2. В выпадающем списке **"From"** выберите `USD - US Dollar`
3. В выпадающем списке **"To"** выберите `EUR - Euro`
4. В поле **"Amount"** введите `100`
5. Нажмите кнопку **"Convert"**

### 💱 Пример 2: Конвертация с копейками/центами

**Задача:** Перевести 1500.75 российских рублей в доллары США.

**Пошаговые действия:**

1. В поле **"From"** выберите `RUB - Russian Ruble`
2. В поле **"To"** выберите `USD - US Dollar`
3. В поле **"Amount"** введите `1500.75`
4. Нажмите **"Convert"**

**💡 Примечание:** Десятичная часть отделяется точкой, а не запятой!

---

### 🔄 Пример 3: Быстрая обратная конвертация с помощью Swap

**Задача:** Сначала узнать курс доллара к йене, а затем быстро посчитать обратный курс.

**Действие 1 - Прямая конвертация:**

1. Выберите `USD → JPY`
2. Введите сумму `1000`
3. Нажмите **"Convert"**

**Действие 2 - Обратная конвертация:**

4. Нажмите синюю кнопку **"⇄"** (Swap)
5. Валюты автоматически поменяются: теперь `JPY → USD`
6. Введите `148250`
7. Нажмите **"Convert"**

**💡 Совет:** Кнопка Swap экономит время, когда нужно быстро сравнить прямой и обратный курс!

## Как получить API-ключ

1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com)
2. Нажмите "Get Free Key"
3. Зарегистрируйтесь, указав email
4. Подтвердите email
5. Получите API-ключ на странице дашборда
6. Замените `YOUR_API_KEY_HERE` в файле `currency_converter.py` на ваш ключ:
